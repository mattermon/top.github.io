重置MySQL密码

2014-07-25
MySQL

会者不难，难者不会。

编辑配置vi /etc/my.cnf

[mysqld]
skip-grant-tables

修改密码

use mysql;
UPDATE user SET Password=password('new-password') WHERE User='root';

重启，并把第一步中增加的选项注释

service mysqld restart
