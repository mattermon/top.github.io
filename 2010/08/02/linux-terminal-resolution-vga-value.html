Linux Terminal Resolution Vga Value

2010-08-02
Linux

虚拟机会按默认的分辨率启动，有时窗口过大或者过小都会影响使用，特别在没有安装GUI桌面的情况下。所以为了解决这个问题，请自动对号入座选择一个十六进制的代码加入到启动文件中，以后系统都将以指定的分辨率启动。同时，此方案同样适用于非虚拟机。

— 320×200 640×400 640×480 800×500 800×600 896×672 1024×640 1024×768 1280×1024
