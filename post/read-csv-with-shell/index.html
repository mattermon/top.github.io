read csv with shell

Development Sep 12, 2016

Use bash while loop and read built-in command;

#!/usr/bin/bash
input="/path/to/csv"
while IFS=',' read -r f1 f2 f3 f4
do
    echo "$f1 $f2"
done < "$input"
